맵 제작 시 주의사항

0. 인자 설명
   - NextStage : 각 노드를 클리어 시 진입할 수 있는 스테이지(다음 스테이지)
	         다음 스테이지에 해당하는 노드를 이곳에 끌어다 놓으면 됨
   - ID : 각 노드가 가지는 고유식별번호이다.(특정 노드에는 지정된 ID가 있음)
   - Type : 노드가 가진 스테이지의 종류이다.
	Battle : 일반, 엘리트, 보스 전투
	Store : 낙인소, 강화소, 탕녀
	Event : 이벤트 및 기타 등등
   - Name : Type에서 더 세분화된 스테이지의 내용
   - StageLevel : 전투 스테이지에서 사용하는 현재 전투의 레벨
   - StageID : 전투 스테이지에서 사용하는 현재 전투의 ID


1. 모든 맵의 맨 앞에는 "StartNode"를 제작한다.
   "StartNode"는 ID를 0으로, Name은 none으로 한다.(나머지는 상관 없음)
   StartNode의 NextStage에 제일 처음 진입해야하는 스테이지를 연결한다.

2. 각 맵의 최종 스테이지는 ID를 99로 한다.
   스테이지가 보스 전투일 경우 최종스테이지인 것으로 하기에는 튜토리얼같은 예외가 있기에 ID를 99일 때로 설정

3. Line의 이미지를 변경하고 싶다면 Prefabs/Stage의 Line의 이미지를 변경하면 됨

4. 맵을 제작하고 싶다면 빈 오브젝트에 노드를 배치하여 맵을 만들고, 그 오브젝트를 Prefabs/Stage/Maps에 넣으면 됨

5. 엘리트, 보스의 노드 이미지는 현재 전투 종류 + _ + StageData.json에 배치된 ID와 같은 수를 넣으면 됨
   ex) 남매의 경우 EliteBattle_0
